# Финансовая модель юнит-экономики маркетплейса

## Бизнес-контекст
Гипотетическая модель для оценки рентабельности привлечения нового продавца на площадку типа Avito. Модель помогает ответить на вопросы: "Сколько мы можем тратить на привлечение продавца?", "Через сколько месяцев он окупится?", "Как рост среднего чека влияет на прибыль?".

## Ключевые метрики (юниты)
*   **Продавец** — основной юнит для расчета.
*   **Покупатель** — косвенный юнит, влияющий на LTV продавца.
*   **Заказ/Сделка** — базовый элемент транзакции.

## Расчетные показатели
1.  **CAC (Customer Acquisition Cost):** Затраты на маркетинг и продажи для привлечения одного продавца.
2.  **LTV (Lifetime Value):** Общий комиссионный доход, который продавец принесет площадке за все время сотрудничества.
    *   *LTV = (Средний месячный оборот × Комиссия %) × Средний срок жизни продавца (в месяцах)*
3.  **PBP (Payback Period):** Время, за который доход от продавца покроет затраты на его привлечение.
4.  **Маржа:** `LTV - CAC`.

## Как работает модель (в Excel/Google Sheets)
Модель построена по принципу **"what-if analysis"** (анализ "что, если"). Пользователь может изменять **"драйверы"** (зеленые ячейки), и все расчеты обновляются автоматически.

**Входные параметры (драйверы):**
*   Бюджет на привлечение продавцов
*   Количество привлеченных продавцов
*   Средний оборот продавца в месяц (руб.)
*   Средняя комиссия площадки (%)
*   Средний срок жизни продавца на площадке (мес.)
*   Операционные расходы на обслуживание (%)

**Выходные результаты:**
*   CAC на одного продавца
*   LTV на одного продавца
*   Срок окупаемости (PBP)
*   Общая прогнозируемая прибыль за период
*   График кумулятивного денежного потока (накопленной прибыли) от когорты продавцов.

## Выводы и инсайты
1.  **Чувствительность:** Модель наиболее чувствительна к изменению **среднего оборота продавца** и **срока его жизни**. Увеличение срока жизни на 20% дает больший прирост LTV, чем увеличение оборота на те же 20%.
2.  **Критическая точка:** При заданных по умолчанию параметрах, CAC не должен превышать **X рублей**, чтобы окупаемость была менее 6 месяцев.
3.  **Рекомендация:** Для роста бизнеса фокус должен быть на **удержании** существующих продавцов (увеличение LTV), а не только на снижении стоимости привлечения (CAC).

## Как пользоваться
1.  Откройте файл `unit_economics_model.xlsx` в Microsoft Excel или Google Sheets.
2.  Измените значения в **зеленых ячейках** (блок "Входные параметры").
3.  Наблюдайте, как автоматически пересчитываются все метрики, графики и итоговые показатели.
4.  Лист `Сценарии` позволяет сравнивать разные стратегии (например, "Агрессивный рост" vs "Рост с прибылью").
